# 実装すべき関数一覧と役割

## 1. 商品一覧表示関連
- `void showProductList()`
  - 商品ID、商品名、価格、在庫数を一覧表示する。
  - 在庫が0の場合は「売り切れ」と表示。

## 2. 投入金額関連
- `int inputMoney()`
  - ユーザーから投入金額を受け取り、正の整数のみを許可。
  - 入力値を返す。

## 3. 商品購入関連
- `void buyProduct()`
  - 商品IDの入力、在庫・金額チェック、商品排出・在庫減少・お釣り計算まで一連の購入処理を行う。

## 4. お釣り管理関連
- `int calcChange(int money, int price)`
  - 投入金額と商品の価格からお釣りを計算し返す。

## 5. 購入履歴関連
- `void savePurchase(const char* productName, int price)`
  - 購入した商品名・価格・日時をCSVファイルに追記。

## 6. メニュー・メインループ
- `void showMenu()`
  - メニュー表示とユーザー入力受付。
- `int main()`
  - プログラムのエントリーポイント。メインループを管理。

## 7. その他（エラーハンドリング等）
- `int getValidInt(const char* prompt)`
  - 整数値の入力を安全に受け取る（数字以外や範囲外の値を弾く）。

---

### 備考
- 商品データは構造体配列で管理。
- 必要に応じて、商品追加・在庫補充・履歴表示などの関数も拡張可能。
- 詳細な処理フローやアルゴリズムは worksheet.md も参照。
